Bot Features & Commands

Vortyx ke saare powerful commands jo aapke server ko next-level banayenge.

🛡️

Moderation

Server ko safe aur clean rakhne ke liye advanced tools.

  • /ban - Kisi user ko server se ban karein.
  • /kick - Kisi user ko server se nikalyein.
  • /mute - Shanti banaye rakhne ke liye mute karein.
  • /warn - Rules todne par warning dein.
  • /clear - Ek saath messages delete karein.
⚙️

Utility

Server ke rozmarra ke kaamon ko aasan banane ke liye.

  • /help - Bot ke saare commands ki list dekhein.
  • /userinfo - Member ki poori details nikalyein.
  • /serverinfo - Server ki saari information dekhein.
  • /ping - Bot ka response time check karein.
  • /avatar - Profile picture full size mein dekhein.
🎫

Ticket System

Members ki help karne ke liye ekdum professional support system.

  • /ticket setup - Server mein support ticket chalu karein.
  • /ticket close - Kaam poora hone par ticket band karein.
  • /ticket add - Kisi ko chalte ticket mein add karein.
🔒

Security

Naye aane wale accounts aur raids se server ko bachane ke liye.

  • /lock - Kisi bhi channel ko turant lock karein.
  • /unlock - Locked channel ko firse open karein.
  • /slowmode - Chat ki speed ko control karne ke liye.
